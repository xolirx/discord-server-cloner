## 📦 Установка

### 🔧 Системные требования
- **Python 3.8** или выше
- **Поддерживаемые ОС:** Windows, Linux, macOS
- **Права администратора** на целевом сервере Discord
- **Discord токен** пользователя с доступом к серверам

### 🚀 Автоматическая установка (Windows)

```bash
1. Скачайте все файлы проекта
2. Запустите `install.bat`
3. Дождитесь завершения установки
```

### 📐 Ручная установка

```bash
# Клонирование репозитория
git clone https://github.com/xolirx/discord-server-cloner.git
cd discord-server-cloner

# Создание виртуального окружения
python -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Активация (Linux/Mac)
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

## ⚙️ Конфигурация

### 🎛️ Настройки задержек

Вы можете настроить параметры в начале файла `channel_copier.py`:

```python
# Параметры задержек (в секундах)
DELAY_BETWEEN_REQUESTS = 3.0    # Задержка между запросами
DELAY_CHANNEL_CREATION = 2.5     # Задержка создания каналов
DELAY_ROLE_CREATION = 3.0        # Задержка создания ролей
DELAY_BULK_OPERATIONS = 5.0      # Задержка массовых операций
```

### 🔧 Рекомендации по настройке

- **Для маленьких серверов** (< 50 каналов) → можно уменьшить задержки на 30%
- **Для больших серверов** (> 200 каналов) → рекомендуется увеличить задержки
- **При ошибках "Rate Limited"** → увеличьте все задержки в 1.5-2 раза

## ❓ Часто задаваемые вопросы

<details>
<summary><b>❔ Можно ли скопировать сообщения?</b></summary>

Нет, Discord API не позволяет программно копировать сообщения между серверами. Это ограничение на уровне платформы.

</details>

<details>
<summary><b>❔ Почему не копируются права доступа?</b></summary>

Права доступа тесно связаны с ролями и участниками. Так как участники не копируются, точное воспроизведение прав невозможно без ручной настройки.

</details>

<details>
<summary><b>❔ Получаю ошибку "403 Forbidden", что делать?</b></summary>

1. Убедитесь, что у вас есть права администратора на **целевом** сервере
2. Проверьте, что токен действителен (не вышел срок)
3. Убедитесь, что целевой сервер существует

</details>

<details>
<summary><b>❔ Как часто можно использовать клонер?</b></summary>

Рекомендуется делать паузы между клонированиями не менее 10-15 минут, чтобы избежать временной блокировки со стороны Discord.

</details>
